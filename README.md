# 동대전고2 기말 파이널 — 인터랙티브 학습 웹앱

2026 1학기 기말 최종점검용. PC·모바일·태블릿 동일하게 동작하는 반응형 단일 웹앱(PWA).

## 핵심 — 선형 학습 코스 (이전/다음으로 쭉)
학생은 **이전/다음 버튼만** 누르며 한 흐름으로 학습합니다:
> 구간 인트로 → 단어 **플래시카드**(눌러서 뜻·동의어·발음) → **미니 퀴즈** → 독해 **지문**(단어 탭 가능) → 독해 **문제** → **구간 완료**(여기서 점수·오답이 선생님 카톡으로 자동 전송)
- 한 구간 ≈ 단어 24개 단위 → 구간을 끝낼 때마다 카톡 전송(자주 피드백).
- 진행 위치는 PIN 로그인으로 서버에 저장 → 어느 기기서나 이어가기.

선형 코스 한 강 흐름: `단어 → 본문 빈칸 → 독해 → 유형별 드릴 → 구간 완료(카톡)`.

## 도구 (메뉴 ≡ 에서)
- **단어장 사전** — 단어 탭 시 뜻·동의어(=)·반의어(↔)·**발음**·출처뱃지·회독·★모르는단어, 검색
- **단어 시험 따로 풀기** — 4종(한글쓰기·영어쓰기·동의어반의어·종합), 객/주관식, 학습/시험 모드
- **유형별 훈련** — 빈칸추론·어법(밑줄 ①~⑤)·순서·삽입·어휘·내용일치 집중 드릴. 문항마다 **💡 단계 힌트 · 📝 풀이 보기 · 🔁 같은 유형 더 풀기**
- **실전 모의고사** — 학교 기말 형식(선택형 20문항·배점 3.0~3.7·빈칸추론 킬러 마지막) 반영. 시험 모드(끝나고 채점)→점수·예상 등급→오답 해설·풀이·힌트
- **데일리 스피드 퀴즈** — 어휘·**어법 OX**·빈칸·연결사 빠른 워밍업
- **본문 빈칸 채우기** — 본문을 떠올리며 빈칸 단답
- **독해 모음** — L0(한국어 선지)~L4(엘리트) 레벨별
- **오답노트** — 모든 오답 누적, 단어 오답 재시험
- **학습 현황 / 설정** — D-day, 스트릭, 다크모드·글자크기, 로그아웃
- 문제 유형 렌더러: MC(5지선다)·**OX**·순서배열·문장삽입·밑줄어법·**cloze 단답**. 전부 자동채점·해설·오답수집·결과 카톡 연결.
- 부가: 오프라인 PWA, 결과 자동 카톡(구간 완료 시)

## 로컬 실행
```
python3 -m http.server 8731 --directory .
# http://localhost:8731  (같은 와이파이의 폰에서 http://<PC IP>:8731 로 접속해 반응형 확인)
```
백엔드 미연결 시 '로컬 모드'로 모든 기능이 동작합니다(카톡·동기화만 제외).
